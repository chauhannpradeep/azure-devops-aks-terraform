## Azure DevOps CI/CD with Terraform, Docker & AKS

This project demonstrates an end-to-end DevOps workflow on Microsoft Azure using
Terraform for infrastructure provisioning, Docker for containerization, and Azure Kubernetes Service (AKS) for orchestration.

### Key Features
- Infrastructure as Code using Terraform
- CI/CD pipelines in Azure DevOps
- Docker image build and push to Azure Container Registry
- Automated deployment to AKS
- Kubernetes rolling updates and health checks
- Monitoring with Azure Log Analytics

### Technologies
Azure DevOps | Terraform | Docker | AKS | Azure | .NET | Kubernetes
